About this website

This website is pretty simple. Right now, I write the HTML for it by hand - no generators. I'm a big fan of various efforts to use HTML's existing qualities for accessibility and design. I think that the web should be designed around simple, handmade things and believe that I can be a part of that.

This website uses MathJax and Highlight.js to render math and code; however, I intend for this site to be readable even without them enabled. Feel free to disable scripts if you like.
